# DigitalMenu

## Description

This is a website like a digital menu for a restaurant.

## Installation

1. clone the repository
2. run `npm install`
3. run `ng serve` to run the project

or paste the following command in your terminal

```bash

git clone
cd DigitalMenu
npm install
ng serve

```
